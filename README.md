​	属性：id、姓名、年龄、家庭住址

​	键盘录入每一个学生信息并添加，需要满足以下要求：

* id唯一

#### 删除功能：

​	键盘录入要删除的学生id，需要满足以下要求：

* id存在删除
* id不存在，需要提示不存在，并回到初始菜单

#### 修改功能：

​	键盘录入要修改的学生id，需要满足以下要求

* id存在，继续录入其他信息
* id不存在，需要提示不存在，并回到初始菜单
